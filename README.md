# Setting
Install `TA-Lib`:
```
# Mac
brew install ta-lib
pip3 install TA-Lib

# Linux
wget http://prdownloads.sourceforge.net/ta-lib/ta-lib-0.4.0-src.tar.gz
tar -xzf ta-lib-0.4.0-src.tar.gz
cd ta-lib
sudo make
sudo make install
pip3 install TA-Lib
```
Install packages:
```
pip3 install pandas
pip3 install numpy
pip3 install mplfinance
```
